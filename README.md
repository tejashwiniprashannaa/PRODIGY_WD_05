# 🌤 Weather App

A simple weather application built using HTML, CSS, and JavaScript that fetches real-time weather data using an API.

## 🚀 Features
- Search weather by city
- Displays temperature, humidity, wind speed
- Shows weather condition and icon
- Loading indicator for better user experience
- Error handling for invalid city input

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## 📸 How It Works
1. User enters a city name
2. App sends request to weather API
3. Receives data in JSON format
4. Displays weather details dynamically on UI

## 🎯 Learning Outcome
- API integration using fetch
- Async/Await handling
- DOM manipulation
- Error handling in JavaScript

## 💻 How to Run
1. Download or clone the project
2. Open `index.html` in browser
3. Enter a city and get weather details
